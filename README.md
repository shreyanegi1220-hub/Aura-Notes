# Aura Notes ✨
### A Personalized Creative Midnight Journal & Task Tracker

**Aura Notes** is a high-performance, aesthetic digital notebook designed for focus, creativity, and organization. Built with a dramatic **Midnight Rose** theme, it combines free-form drawing, advanced shape toolkits, and a built-in task tracking system.

## 🌟 Key Features
-   🌑 **Dramatic Midnight Theme**: A stunning, high-contrast visual experience optimized for focus.
-   🖋️ **Creative Toolkit**: Full-featured drawing engine with shapes (Rectangles, Circles, Triangles, Stars) and a precision Eraser.
-   📏 **Ruler Alignment**: Writing on lined pages automatically snaps to the grid for a perfect, clean look.
-   📋 **Tasks & Tracking**: A dedicated sidebar for your daily To-Do list, separate from your main notes.
-   📱 **Installable PWA**: Install Aura Notes directly on your phone, tablet, or desktop.
-   ☁️ **GitHub Pages Ready**: Host your personal journal for free and access it from any device.

## 🚀 How to Use
1.  **Text Mode**: Click "Text" to start writing. Use **Bold**, **Bullets**, and **Ticks** from the top toolbar.
2.  **Draw Mode**: Click "Draw" to sketch. Use the floating toolbar to change colors, brush size, or switch to shapes.
3.  **Organization**: Create different "Notebooks" (Personal, Work, etc.) to categorize your journeys.
4.  **Tracking**: Use the sidebar "Tasks & Tracking" section to manage your immediate daily goals.

## 🛠️ Technical Details
-   **Core**: HTML5, CSS3, JavaScript (ES6+).
-   **Icons**: [Lucide Icons](https://lucide.dev/).
-   **Typography**: Google Fonts (Outfit, Comfortaa, Caveat).
-   **Icons**: PWA-ready with `manifest.json` and `sw.js`.
